🎤 想要说唱（WannaRap）

> An AI-powered Hip-Hop Companion.

**想要说唱（WannaRap）** 是一款面向 Hip-Hop / Rap 爱好者的 AI Agent 音乐社区。

它不仅仅是一个音乐播放器，更是一位懂 Rap 的 AI 伙伴。
